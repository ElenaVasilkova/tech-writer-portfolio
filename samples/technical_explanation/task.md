# Тестовое задание: Объяснение условий группировки профилей

**Источник:** Тестовое задание на позицию технического писателя  
**Дата выполнения:** 2025

## Постановка задачи

> **Задача:** Объяснить пользователям условия группировки профилей 
> в разных социальных сетях.

**Исходный текст для анализа:**

Яндекс группирует профили пользователя в разных социальных сетях, если:  
• Два профиля взаимно ссылаются друг на друга.  
• Один профиль ссылается на другой с помощью подтвержденной ссылки.  
Подтвержденной считается ссылка, для выставления которой требуется
ввести пароль другой сети.
