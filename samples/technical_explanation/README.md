# Объяснение условий группировки профилей

**Задача:** Объяснить пользователям сложное техническое условие 
«подтверждённая ссылка» и дать практические инструкции по объединению профилей.

**Проблема исходного текста:**
- Формулировка «требуется ввести пароль другой сети» непонятна рядовому пользователю
- Нет инструкций «что делать»
- Не указаны ограничения

**Решение:**
1. **Переформулировала условие**  («пройти авторизацию» вместо «ввести пароль»)
2. **Добавила практические инструкции** для двух основных сценариев (через Яндекс и вручную)
3. **Указала конкретные пути** в каждой соцсети с ссылками
4. **Предупредила** об ограничениях (Одноклассники, индексация)
5. **Добавила канал обратной связи** (служба поддержки)
